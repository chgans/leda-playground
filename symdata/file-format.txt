- circle:
  - center X/Y
  - radius
  - thickness
  - fill

- Text:

  - Pos X/Y

- Pin:
  - orientation: Up/Down/Right/Left
  - pin name (~ if no name)
  - pin number
  - length
  - pin name size
  - pin number size
  - electric type
  - shape (clock, inverted, ...)
  - shape visible
